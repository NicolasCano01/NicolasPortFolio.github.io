Data cleaning and data manipulation

Data cleaning, in a construction company

Problem

The company has an old ERP system, that exports the data in a difficult format to read and understand.
The objective is to normalize the data and store in a table ready to be processed.

Action taken

Find a pattern on the source files and through VBA, create a macro that could be replicated in all files. Power Query needs to be involved in the process to override column dynamization.

Outcome

A clean and simple table, easy to read and usable for reporting purposes.

Data Visualization

Basket Analysis

Problem

The company is planning to buy one of their suppliers.
The objective is to show each supplier's income and which supplier's items paired better with the company's products.

Action taken

Using Power Pivot and Power BI , create a data model and show visually the total of sales made by each supplier and what percentage represent of the company's income.
On the second sheet, using the most popular company's items, find the products (and supplier) better paired with them.

Outcome

Two different results, the supplier with the highest income is different from the supplier with items that paired better with the company's products.

Quantity of Sales Analysis

Problem

The company lacks a way of visualising the current sales performance against the previous year and previous month.

Action taken

Using Power Pivot and Excel Pivot Tables, get the summary information of the quantities and also presenting the user the option to see a breakdown by staff member and type of sale.

Outcome

A Dashboard was created with a simple graphic, along with a few tables with more details and a different range of slicers to help the user filter the Dashboard information accordingly.
